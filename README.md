# PathWise AI

**PathWise AI: Personalized Career & Life Companion** is an interactive front-end hackathon prototype built for students, graduates, and job seekers who need support with interview preparation, resume building, career discovery, and wellbeing awareness.

This project was created by **Team Nextara** as a polished product-style demo for hackathon presentation and concept validation.

## Overview

Many young people experience stress, confusion, and low confidence when preparing for careers and job applications. PathWise AI is designed as a single demo platform that combines:

- interview practice
- resume support
- career recommendations
- simple wellbeing guidance

The current version is a **front-end prototype only**. It is meant for offline demonstration, UI presentation, and hackathon judging, while showing how a fuller AI-powered platform could work in the future.

## Main Objectives

- Reduce stress in job seeking through guided support
- Provide interactive interview coaching with real-time feedback
- Help users build professional resumes with dynamic preview
- Recommend suitable career paths based on profile fit
- Encourage balanced career growth with wellbeing-aware guidance

## Current Prototype Features

### 1. AI Interview Practice

The interview section is designed like a live AI practice panel with:

- **HR AI** for behavioral and personality questions
- **Technical AI** for technical and problem-solving questions
- **Manager AI** for leadership and real-world scenario questions

Interactive demo capabilities include:

- interviewer role switching
- browser voice playback
- text or microphone answer input
- live confidence, clarity, and relevance feedback
- role-based questioning flow
- end-of-session interview report
- optional real camera preview in supported browsers

### 2. AI Resume Builder

The resume section lets users enter:

- full name
- target role
- email
- phone
- location
- portfolio / LinkedIn
- university / institution
- education
- professional summary
- skills
- work experience
- projects
- certifications
- achievements / leadership

The page then updates a **live CV preview** and supports **print / PDF-style demo export** from the browser.

### 3. Career Recommendation System

The career recommendation module checks broader profile fit using:

- skills
- interests
- university / institution
- field of study
- personality style
- stress tolerance
- preferred work style
- preferred work environment
- leadership preference
- communication strength
- creativity level
- data / numbers comfort

The current version can display **up to 14 career suggestions** with:

- match score
- short explanation
- improvement tips

### 4. Wellbeing Support

The wellbeing section includes a richer check-in interaction using:

- current stress level
- sleep hours
- main pressure source
- energy level
- support level

Based on those inputs, the prototype gives:

- supportive status messaging
- non-clinical encouragement
- personalized practical tips
- a more positive, friendly user experience during career preparation

### 5. Dashboard

The dashboard presents quick product-style metrics for:

- interview score
- resume completion
- career match
- wellbeing status

This helps the demo feel like a complete product rather than a standalone tool.

### 6. Team and Project Showcase

The team section now includes:

- member roles
- demo email contacts
- MSU affiliation for each member
- mentor details
- simple GitHub project cards linked to the team GitHub profile and PathWise AI repository

### 7. Local Demo Login Flow

The project includes a lightweight browser-based login/register flow for demo use only.

It supports:

- email-first login check
- local registration
- local sign-in
- local account persistence with browser storage

This is only for prototype realism and is **not a production authentication system**.

## Design Direction

The UI is intentionally styled like a **modern startup SaaS demo**, not an academic poster page.

Design goals:

- purple and white visual identity
- soft gradients and rounded cards
- dark premium interview workspace
- responsive layout for laptops, tablets, and smartphones
- product-style sectioning
- smooth transitions and hover effects
- clean visual hierarchy for hackathon judging

## Technology Stack

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**

JavaScript is used for:

- interview session logic
- question switching
- voice playback
- microphone capture
- camera preview handling
- dynamic resume preview
- career scoring and rendering
- wellbeing interaction
- local demo authentication flow

CSS media queries are used to keep the layout adaptive across larger laptop screens and smaller smartphone screens.
